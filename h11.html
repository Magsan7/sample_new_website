What is External CSS?

External CSS means the style rules live in a separate styles.css file and are linked from your HTML using <link>. This keeps markup clean and styles reusable.

Selectors

Element, class and ID selectors are used to target different parts of the page.

ID vs Class

An ID is unique (used for one element). Classes can be reused across many elements.

Pseudo-classes

Hover, focus and other pseudo-classes let you style interactive states.
